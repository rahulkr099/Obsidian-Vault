# Contract Driven API for a Todo App

## Base URL

```
/api/v1
```

---

# 1. Create Todo

### Endpoint

```
POST /todos
```

### Description

Create a new todo item.

### Request Body

```json
{
  "title": "Learn Express",
  "description": "Build REST APIs",
  "tags": ["study"]
}
```

### Validation

|Field|Type|Required|
|---|---|---|
|title|String|Yes|
|description|String|No|
|tags|Array|No|

### Success Response

Status Code

```
201 Created
```

Body

```json
{
  "success": true,
  "data": {
    "_id": "65fd123abc",
    "title": "Learn Express",
    "description": "Build REST APIs",
    "status": "pending",
    "tags": ["study"],
    "createdAt": "2026-03-12T10:00:00Z"
  }
}
```

---

# 2. Get All Todos

### Endpoint

```
GET /todos
```

### Description

Return list of all todos.

### Query Parameters

|Param|Type|Description|
|---|---|---|
|page|Number|page number|
|limit|Number|results per page|
|status|String|filter by status|
|sort|String|sort fields|

Example

```
GET /todos?page=1&limit=10&status=pending
```

### Success Response
Status Code:
```
200 OK
```

Body:
```json
{
  "success": true,
  "results": 2,
  "data": [
    {
      "_id": "123",
      "title": "Learn Node",
      "status": "pending"
    },
    {
      "_id": "124",
      "title": "Practice DSA",
      "status": "completed"
    }
  ]
}
```

---

# 3. Get Single Todo

### Endpoint

```
GET /todos/:todoId
```

Example

```
GET /todos/65fd123abc
```

### Description
Return a single todo by todoId
### Success Response

```
200 OK
```

```json
{
  "success": true,
  "data": {
    "_id": "65fd123abc",
    "title": "Learn Express",
    "description": "Build APIs",
    "status": "pending"
  }
}
```

### Error Response
Status Code:
```
404 Not Found
```
Body:
```json
{
  "success": false,
  "message": "Todo not found"
}
```

# 6. Standard Error Format (Very Important)

Good APIs keep **same error structure everywhere**.

Example:

```json
{
  "success": false,
  "message": "Validation failed",
  "errors": [
    {
      "field": "title",
      "message": "Title is required"
    }
  ]
}
```
