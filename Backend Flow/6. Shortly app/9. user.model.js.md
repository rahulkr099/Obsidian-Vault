# 4️⃣ Field 1 — firstName

### Your Code

```javascript
firstName: {
 type: String,
 required: true,
 trim: true
}
```

### Pseudocode

```text
STORE user's first name
REQUIRED field
REMOVE extra spaces automatically
```

Example stored value:

```json
"firstName": "Rahul"
```

If user enters:

```text
"  Rahul   "
```

Database stores:

```text
Rahul
```

# 7️⃣ Field 4 — Password

### Your Code

```javascript
password:{
 type:String,
 required:function(){
   return this.authType !== 'google'
 }
}
```

### Pseudocode

```text
STORE hashed password
REQUIRED only for local authentication
NOT required for Google login
```

Example:

```text
Local signup
password = "$2b$10$dfh23k..."
```

Google login:

```text
password = null
authType = "google"
```

# 1️⃣5️⃣ Practical Example — Signup Flow

### Step 1 — User registers

Request:

```http
POST /api/v1/signup
```

Body:

```json
{
 "firstName":"Rahul",
 "lastName":"Kumar",
 "email":"rahul@gmail.com",
 "password":"123456",
 "role":"user"
}
```

---

### Step 2 — Controller hashes password

```text
123456 → bcrypt → hashed password
```

Example:

```text
$2b$10$98shd83...
```

---

### Step 3 — Create database record

```javascript
User.create({...})
```

Stored document:

```json
{
 "_id":"65fa12abc",
 "firstName":"Rahul",
 "lastName":"Kumar",
 "email":"rahul@gmail.com",
 "password":"$2b$10$98shd83...",
 "authType":"local",
 "role":"user"
}
```

---

# 1️⃣6️⃣ Practical Example — Login Flow

```http
POST /login
```

Body:

### Step 2 — Database lookup


---

### Step 3 — Password verification

```text
bcrypt.compare(password, hashedPassword)
```

### Step 4 — Tokens generated

```text
refreshToken
```

Database updated:

```json
{
 "refreshToken":"eyJhbGciOiJIUzI1NiIs..."
}
```
