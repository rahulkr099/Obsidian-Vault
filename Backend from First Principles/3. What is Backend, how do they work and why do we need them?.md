# What is Backend? 
Backend is a computer which is listening for HTTP or websocket or Grpc or any other kind of request through an open port which is accessible over the Internet so that clients or other frontends can connect to it, send data to it, receive data to it.

## What is server?
A server is a computer or program that provides data, services, or resources to other computers (Clients) over a network.

We call it server because it provides or it serves some kind of content whether it is static files like images, JS files, HTML files or it could be JSON and it also accept data if the client sends something.
