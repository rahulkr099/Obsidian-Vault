##  **INITIAL SETUP**

```
DISPLAY a search input box
DISPLAY a list of items
```

---

## **PROCEDURE filterItems()**

```
READ text entered in search box
CONVERT search text to lowercase and trim it

GET all list items

FOR each item in items
    READ text of item
    CONVERT text to lowercase
    CHECK if text contains filter
    IF it matches
        SHOW item
    ELSE
        HIDE item
END FOR

```

---

## **USER INTERACTION**

```
WHEN user types in the search box
    CALL filterItems()
```

## **END**
