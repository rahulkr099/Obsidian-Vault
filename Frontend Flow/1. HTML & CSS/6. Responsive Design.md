## Step 1: Learn Mobile-First CSS (MOST IMPORTANT)

### Write base CSS for mobile (no media query)

```css
body {
    font-size: 14px;
}

.container {
    padding: 1rem;
}

.card {
    width: 100%;
}
```

This is for **small screens by default**.

---

### Then add media queries for bigger screens

```css
@media (min-width: 768px) {
    body {
        font-size: 16px;
    }

    .container {
        max-width: 720px;
        margin: auto;
    }
}

@media (min-width: 1024px) {
    body {
        font-size: 18px;
    }
}
```

‚úî Much cleaner  
‚úî Fewer overrides  
‚úî Professional style

## Step 2: Stop using px everywhere ‚ùå

This is where beginners struggle.

### Use these units instead:

|Use for|Unit|
|---|---|
|Font|`rem`|
|Spacing|`rem`|
|Width|`%`, `max-width`|
|Layout|`flex`, `grid`|

### Example

```css
body {
    font-size: 1rem; /* scales automatically */
}

.card {
    padding: 1rem;
    margin-bottom: 1rem;
}
```

Now when font size changes ‚Üí everything scales ‚ú®

## Step 3: Learn Flexbox (90% of responsive layout)

If you learn **only one thing**, learn this.

```css
.container {
    display: flex;
    flex-direction: column;
    gap: 1rem;
}
```

### Change layout on bigger screens

```css
@media (min-width: 768px) {
    .container {
        flex-direction: row;
    }
}
```

üì± Mobile ‚Üí stacked  
üíª Desktop ‚Üí side-by-side

This pattern solves **most layouts**.

---

## Step 4: Use Grid only when needed

Grid is great for cards, galleries, dashboards.

```css
.cards {
    display: grid;
    grid-template-columns: 1fr;
    gap: 1rem;
}

@media (min-width: 768px) {
    .cards {
        grid-template-columns: repeat(2, 1fr);
    }
}

@media (min-width: 1024px) {
    .cards {
        grid-template-columns: repeat(3, 1fr);
    }
}
```

Simple. Predictable. Powerful.

## Step 5: Learn these breakpoints (don‚Äôt overthink)

Use **only these 3** at start:

```css
/* tablet */
@media (min-width: 768px) {}

/* laptop */
@media (min-width: 1024px) {}

/* desktop */
@media (min-width: 1280px) {}
```

That‚Äôs enough for 95% of sites.
