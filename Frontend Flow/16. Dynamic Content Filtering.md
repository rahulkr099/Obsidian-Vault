# **Pseudocode: Content Filtering (Search Filter)**

---

## **START**

---

## **INITIAL SETUP**

```
DISPLAY a search input box
DISPLAY a list of items
```

---

## **PROCEDURE filterItems()**

```
READ text entered in search box
CONVERT search text to lowercase

GET all list items

FOR each item in the list
    GET item text
    CONVERT item text to lowercase

    IF item text contains search text
        SHOW the item
    ELSE
        HIDE the item
    END IF
END FOR
```

---

## **USER INTERACTION**

```
WHEN user types in the search box
    CALL filterItems()
```

---

## **END**

---
