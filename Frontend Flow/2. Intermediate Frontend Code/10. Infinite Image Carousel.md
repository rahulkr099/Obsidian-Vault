### Show Slide Function

```
FUNCTION showSlide(index)
    MOVE carousel using transform based on index
END FUNCTION
```

---

### Next Slide Logic

```
FUNCTION nextSlide
    INCREMENT currentIndex

    IF currentIndex exceeds last slide
        RESET currentIndex to 0
    END IF

    CALL showSlide(currentIndex)
END FUNCTION
```

---

### Previous Slide Logic

```
FUNCTION prevSlide
    DECREMENT currentIndex

    IF currentIndex is less than 0
        SET currentIndex to last slide
    END IF

    CALL showSlide(currentIndex)
END FUNCTION
```

---

### Button Controls

```
WHEN Next button is clicked
    CALL nextSlide
END WHEN

WHEN Previous button is clicked
    CALL prevSlide
END WHEN
```

---

### Auto Sliding
