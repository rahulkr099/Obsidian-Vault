## 2Ô∏è‚É£ CSS File ‚Äì Layout & Animation (styles.css)

### Pseudocode ‚Äì CSS

```

```

‚ú® **Key idea**

## 3Ô∏è‚É£ JavaScript File ‚Äì Page Switching Logic (script.js)

**Goal:**  
When user clicks a link ‚Üí hide current page ‚Üí show target page.

### Pseudocode ‚Äì JavaScript

```

GET all page sections

FOR each navigation link
    ADD click event listener

    WHEN link is clicked
        PREVENT page reload

        GET target section name from clicked link

        FOR each page
            REMOVE "active" class

        FIND the page that matches target name
        ADD "active" class to that page
```

üöÄ **What‚Äôs happening**

- JavaScript does **not reload** the page
    
- It only switches CSS classes
    
- CSS handles the animation automatically
