# **Pseudocode: Autocomplete / Search Suggestions**

---

## **START**

---

## **INITIAL SETUP**

```
CREATE a search input box
CREATE an empty list to display suggestions
STORE predefined suggestion words in a list
```

---

## **PROCEDURE autocomplete()**

```
READ text entered in the search box
CONVERT input text to lowercase and trim it

CLEAR all previous suggestions from the list

FILTER each item in suggestions
    CONVERT item to lowercase
    CHECK if item contains user input
	Use FOR EACH which will work if word exists
        CREATE a new list item
        SET list item text to the word
        ADD list item to suggestions list
    END IF
END FOR
```

---

## **USER INTERACTION**

```
WHEN user types in the search box
    CALL autocomplete()
```

## **END**
