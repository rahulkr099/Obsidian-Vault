## üìå PSEUDOCODE ‚Äî Fetch Users using async / await (with error handling)

```
START

1. DEFINE an asynchronous function named fetchUsers

2. INSIDE fetchUsers
   a. TRY
      i.   SEND request to API URL using fetch
           ‚Üí "https://jsonplaceholder.typicode.com/users"

      ii.  WAIT until response is received

      iii. CONVERT response to JSON format
           ‚Üí WAIT until JSON data is ready

      iv.  FIND the <ul> element with id "users"
```

## üß† What async / await changes (easy explanation)

- `await` ‚Üí **pause execution until result arrives**
    
- Code looks **top-to-bottom**, like normal logic
    
- Easier to read than `.then().then()`
    

This is why modern JavaScript prefers async / await.

## üéØ Interview-ready explanation (short & strong)

> I use an async function to fetch data from the API.  
> I wait for the response and convert it into JSON.  
> Then I loop through the data and display each user inside a list.  
> I also use try-catch to handle errors safely.

---

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Fetch Users</title>

    <style>
        body {
            font-family: Arial, sans-serif;
            padding: 1rem;
        }

        .loading {
            color: blue;
            font-weight: bold;
        }

        .error {
            color: red;
            font-weight: bold;
        }

        ul {
            padding-left: 1rem;
        }
    </style>
</head>
<body>

    <h2>Users</h2>
    <p id="status" class="loading">Loading users...</p>
    <ul id="users"></ul>

    <script>
        async function fetchUsers() {
            const statusText = document.getElementById("status");
            const userList = document.getElementById("users");

            try {
                // Fetch data from API
                const response = await fetch("https://jsonplaceholder.typicode.com/users");

                // Check if response is OK
                if (!response.ok) {
                    throw new Error("Network response was not OK");
                }

                // Convert response to JSON
                const data = await response.json();

                // Clear loading message
                statusText.textContent = "";

                // Loop through users and add to list
                data.forEach(user => {
                    const li = document.createElement("li");
                    li.textContent = user.name;
                    userList.appendChild(li);
                });

            } catch (error) {
                // Show error message
                statusText.textContent = "Failed to load users";
                statusText.className = "error";
                console.error(error);
            }
        }

        // Call function
        fetchUsers();
    </script>

</body>
</html>

```
