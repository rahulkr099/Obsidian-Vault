# üß† `script.js` ‚Äî MAIN LOGIC (VERY IMPORTANT)

This file controls **lazy loading behavior**.

```text
START script as ES module

GET reference to button using ID "loadModuleBtn"

GET reference to output div using ID "moduleOutput"

ADD click event listener to button

WHEN button is clicked
  DO the following steps:

  TRY
    DYNAMICALLY import "lazyModule.js"
      // JavaScript does NOT load this file at page load
      // It loads only when this line runs

    WAIT until module is fully loaded

    EXTRACT function from imported module
      (example: showMessage)

    CALL the imported function
      STORE returned value or result

    DISPLAY result inside moduleOutput div

  CATCH error (if module fails to load)
    DISPLAY error message in console
    SHOW user-friendly error in UI
END
```

üìå **Big idea (INTERVIEW GOLD ‚≠ê)**

- `import()` runs **only when needed**
    
- Reduces initial JavaScript load
    
- Improves performance
