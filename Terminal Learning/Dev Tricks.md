### ✔ VS Code–style Emmet

Type:

`div.container>ul>li*5`

Press:

`Ctrl + y ,   (yes, release Ctrl before pressing comma)`

You get:

`<div class="container">   <ul>     <li></li>     <li></li>     <li></li>     <li></li>     <li></li>   </ul> </div>`

- HTML
    
- CSS
    
- React (JSX)
